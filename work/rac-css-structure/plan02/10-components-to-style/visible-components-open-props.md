# Open Props Visible Surface

Open Props is not a component library, so there is no public component catalog to list.

The closest public building blocks are its token and bundle surfaces:

- Token packs: `colors`, `gradients`, `shadows`, `aspect ratios`, `typography`, `easings`, `animations`, `sizes`, `borders`, `z-indexes`, `media-queries`
- Theme and reset bundles: `normalize.min.css`, `normalize.light.css`, `normalize.dark.css`, `theme.light.css`, `theme.dark.css`
- Optional opinionated bundles: `buttons.min.css` and related light/dark variants
- Token export formats: CSS, PostCSS, JSON, and JavaScript
- Scoped and prefixed builds: shadow-DOM-scoped imports and `--op-*` prefixed builds

If you need a literal component inventory, the answer is: none.
