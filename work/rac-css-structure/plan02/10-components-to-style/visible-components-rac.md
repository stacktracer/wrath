# RAC Visible Components

This inventory covers `react-aria-components@1.16.0` as installed in this repo. It includes public RAC components that own a visual surface, including visible subparts that RAC exposes directly for styling.

Not included:

- `Autocomplete`
- `Collection`
- `DialogTrigger`
- `FileTrigger`
- `Focusable`
- `MenuTrigger`
- `Pressable`
- `Section`
- `SharedElement`
- `SharedElementTransition`
- `SubmenuTrigger`
- `TooltipTrigger`
- `Virtualizer`
- `VisuallyHidden`
