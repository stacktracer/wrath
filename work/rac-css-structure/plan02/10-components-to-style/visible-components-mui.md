# MUI Component Inventory

This inventory covers Material UI core, meaning `@mui/material` and `@mui/material/*`, not MUI X, Base UI, or Joy UI. It is based primarily on the official [All Components](https://mui.com/material-ui/all-components/) page and the package barrel at [packages/mui-material/src/index.js](https://raw.githubusercontent.com/mui/material-ui/master/packages/mui-material/src/index.js).

The docs page also shows `Number Field` and `Transfer List`, but the current `@mui/material` barrel does not export those names, so they are not listed here.
