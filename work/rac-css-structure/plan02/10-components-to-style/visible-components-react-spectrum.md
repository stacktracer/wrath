# React Spectrum Visible Components

This inventory covers the current React Spectrum 2 surface on `react-spectrum.adobe.com`, which is the docs surface now centered around `@react-spectrum/s2`.

Legacy React Spectrum v3 still exists as a separate surface and includes additional or older APIs such as `DialogContainer`, `Content`, and `Footer`. Those are not listed here.
