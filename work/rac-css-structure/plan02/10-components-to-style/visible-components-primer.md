# Primer Visible Components

This inventory covers Primer's public Product UI component surface, centered on `@primer/react` and the Product UI catalog at `primer.style/product/components/`.

Not included:

- Primer Rails components
- internal components
- React hooks
- CSS utilities
- primitives

## Documented But Private

- Overlay

Primer documents `Overlay` in the public component catalog, but the page explicitly says it is an internal/private API and should be used only as a base for other components.
